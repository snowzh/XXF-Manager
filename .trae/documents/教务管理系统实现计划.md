# 中小学生教务管理系统开发计划（Docker版）

## 项目概述

使用Django+bootstrap开发一个针对中小学生的教务管理系统，包含用户认证、学生管理、教师管理、课程管理、成绩管理等核心功能。

## 技术栈

* **后端**：Django 4.2

* **前端**：Bootstrap 5

* **数据库**：MySQL（使用Docker容器）

* **MySQL驱动**：pymysql（替代mysqlclient）

* **认证**：Django自带认证系统

* **容器化**：Docker

## 核心功能模块

### 1. 用户认证模块

* **角色**：管理员、教务老师、教学老师

* **功能**：登录、密码重置、角色权限控制

### 2. 学生管理模块

* 学生信息录入、编辑、删除

* 班级分配

* 学生档案管理

### 3. 教师管理模块

* 教师信息录入、编辑、删除

* 科目分配

* 教师档案管理

### 4. 班级管理模块

* 班级创建、编辑、删除

* 班主任分配

* 班级学生列表

### 5. 课程管理模块

* 课程创建、编辑、删除

* 课程表生成

* 课程分配给班级和教师

### 6. 成绩管理模块

* 成绩录入、编辑、删除

* 成绩统计与分析

* 成绩单生成与导出

### 7. 考勤管理模块

* 考勤记录录入

* 考勤统计

* 请假管理

### 8. 通知管理模块

* 校园公告发布

* 班级通知

* 个人消息

## 角色权限

### 管理员

* 拥有所有权限

* 可以管理所有用户

* 可以配置系统设置

### 教务老师

* 可以管理学生信息

* 可以管理班级信息

* 可以管理课程信息

* 可以查看所有成绩和考勤

### 教学老师

* 可以查看所教班级的学生信息

* 可以录入和查看所教课程的成绩

* 可以录入和查看所教班级的考勤

* 可以发布班级通知

## 开发步骤

1. **项目初始化**

   * 创建Django项目

   * 配置虚拟环境

   * 安装必要依赖（包括pymysql）

2. **配置Docker环境**

   * 安装Docker

   * 创建docker-compose.yml文件

   * 配置MySQL容器

3. **创建应用**

   * 创建users、students、teachers、courses、scores、attendance、notifications应用

   * 配置应用路由

4. **设计数据库模型**

   * 设计用户模型（扩展Django自带User模型，添加角色字段）

   * 设计学生、教师、班级、课程、成绩、考勤、通知模型

5. **实现用户认证**

   * 自定义用户认证

   * 角色权限控制

   * 登录、密码重置功能

6. **开发前端页面**

   * 基于Bootstrap 5设计响应式页面

   * 创建base.html模板

   * 开发各模块的CRUD页面

7. **实现各功能模块**

   * 学生管理

   * 教师管理

   * 班级管理

   * 课程管理

   * 成绩管理

   * 考勤管理

   * 通知管理

8. **测试与优化**

   * 功能测试

   * 性能优化

   * 安全性检查

## Docker配置

```yaml
# docker-compose.yml
version: '3.8'

services:
  mysql:
    image: mysql:8.0
    container_name: xxf_manager_mysql
    restart: always
    environment:
      MYSQL_ROOT_PASSWORD: password
      MYSQL_DATABASE: xxf_manager
      MYSQL_USER: django
      MYSQL_PASSWORD: django
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql
    command: --default-authentication-plugin=mysql_native_password

volumes:
  mysql_data:
```

## 数据库配置示例

```python
# settings.py
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'xxf_manager',
        'USER': 'django',
        'PASSWORD': 'django',
        'HOST': 'localhost',
        'PORT': '3306',
        'OPTIONS': {
            'charset': 'utf8mb4',
            'use_unicode': True,
        },
    }
}

# 使用pymysql替代MySQLdb
import pymysql
pymysql.install_as_MySQLdb()
```
